

SCRIPT DE INICIALIZACIÓN Y CONFIGURACIÓN DEL SISTEMA PARA CENTOS 6
==================================================================


- Muévete al directorio donde se encuentra este fichero:
`# cd path/to/folder`


- Asegúrate de ejecutar el script como **root**, para ello, utiliza una de las dos opciones siguientes:
`# sudo su`
ó
`# su root`


-Ejecuta el script en bash de nombre *setup.sh* y atiende a los datos que pida por pantalla:
`# ./setup.sh`
